## ğŸš€ Getting Started

### Prerequisites

- **Java Development Kit (JDK)** 8 or higher
- Any IDE (VS Code, IntelliJ IDEA, Eclipse) or command line

### Installation

```bash
# Clone the repository
git clone https://github.com/cloud-rooted/temp.git

# Navigate to project directory
cd temp/external-lab
```

### Running a Program

```bash
# Navigate to the desired set folder
cd external-lab/set-01

# Compile the Java file
javac HashSetDemo.java

# Run the compiled program
java HashSetDemo
```

### Quick Run (Single Command)

```bash
# Compile and run in one line
cd external-lab/set-03 && javac ExceptionDemo.java && java ExceptionDemo
```

### Running JDBC Programs

```bash
# Make sure MySQL Connector JAR is in classpath
cd external-lab/set-01
javac -cp ".;mysql-connector-java.jar" JDBCDemo.java
java -cp ".;mysql-connector-java.jar" JDBCDemo
```

### Running Servlet Programs

```bash
# Requires Apache Tomcat servlet container

# 1. Start Tomcat (Windows)
cd external-lab/set-14/apache-tomcat/bin
startup.bat

# 2. Ensure servlet structure is correct:
#    apache-tomcat/webapps/MyApp/WEB-INF/classes/HelloServlet.class
#    apache-tomcat/webapps/MyApp/WEB-INF/web.xml

# 3. Access via browser:
#    http://localhost:8080/MyApp/hello
```

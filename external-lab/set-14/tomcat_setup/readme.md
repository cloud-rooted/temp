# Apache Tomcat Setup (Quick Guide)

## Prerequisites
- Java JDK installed
- Apache Tomcat extracted in the project folder

## Start Tomcat
### Windows
```bat
cd apache-tomcat/bin
startup.bat
```


open 
http://localhost:8080/MyApp/hello


# Servlet files
Run Servlet
Ensure servlet structure:
apache-tomcat/webapps/MyApp/WEB-INF/classes/HelloServlet.class
apache-tomcat/webapps/MyApp/WEB-INF/web.xml
